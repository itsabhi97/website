### Overview
For a media planning software company, I led a cross-functional team of 3 developers and a data scientist to build the frontend, backend, and algorithm entirely from scratch, migrating away from their legacy tech stack.

### Key Contributions
- Designed the database schema capable of storing both raw inputs and outputs from the algorithm responsible for optimal media plans.
- Architected and built the application using Flask (backend) and Material UI (frontend).
- Defined priority matrices mapping business tasks to the engineering sprints.
- Led rigorous code reviews and quality assurance processes across the team.
- Built the foundational predictive algorithm generating optimal media plans based on dynamic criteria mapping.
- Created pre-commit hooks to strictly guard the repository coding standards, reducing reviewer back-and-forth overhead by over 30%.
