### Overview
For a company providing solutions in Cyber Compliance, I led the migration from an old architecture to a new highly-scalable multi-tenancy architecture.

### Key Contributions
- Identified suitable databases to support the new multi-tenancy architecture.
- Merged multiple databases into one and redesigned the database schema from scratch.
- Designed and built pipelines in Databricks using Pyspark to sync data from the old database to the new.
- Built a pipeline in Databricks using Pyspark to sync data from Elasticsearch to the new database.
- Built Streaming Pipelines in Pyspark to ingest data from Kinesis into Databricks and PostgresSQL databases.
